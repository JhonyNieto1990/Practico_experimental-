## 🛠️ Tecnologías y Arquitectura
Este sistema fue desarrollado utilizando:
* **Backend:** Python / Flask
* **Base de Datos:** MySQL (Relacional)
* **Frontend:** HTML5, CSS3 (Bootstrap 5)
* **Seguridad:** Flask-Login para autenticación de usuarios.

La estructura del código sigue un patrón modular:
- `models/`: Definición de tablas y entidades.
- `services/`: Lógica de negocio y manejo de datos.
- `templates/`: Interfaz de usuario dinámica con Jinja2.
- `static/`: Recursos visuales y estilos de Crunch House.
